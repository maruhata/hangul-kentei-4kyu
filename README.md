# ハングル能力検定 合格教材 🇰🇷

ブラウザで学べるインタラクティブなハングル能力検定の学習教材です。インストール不要、URLを開くだけ。**5級**と**4級**を1つのサイトにまとめています。

## 🌐 公開URL

**https://maruhata.github.io/hangul-kentei-4kyu/**

トップページで受けたい級を選べます。
- 5級（入門）… `/5kyu/`
- 4級（やさしい日常会話）… `/4kyu/`

## ✨ 各級の収録内容

| タブ | 内容 |
|---|---|
| 🏠 学習ガイド | 学習計画＋進捗ダッシュボード |
| ㉠ 文字 | 母音・子音・濃音の反切表（タップで🔊音声） |
| 🔊 発音 | パッチムの音・連音化などの読み方 |
| 📘 文法 | 級に応じた文法（5級=基礎13項目／4級=26項目） |
| 🗂 単語帳 | フラッシュカード（5級 約350語／4級 約920語・音声つき） |
| ✏️ クイズ | 6分野の選択式クイズ（採点・自己ベスト保存） |
| 📝 模擬試験 | 本番形式：聞きとり40点＋筆記60点＝100点（60点で合格判定） |

## 🔊 音声について

ブラウザの音声読み上げ機能（Web Speech API, `ko-KR`）を使用しています。
韓国語の音声が端末に入っていない場合、再生されない・英語読みになることがあります。
Windows の場合は「設定 → 時刻と言語 → 言語 → 韓国語を追加（音声付き）」で改善します。

## 📁 構成

```
index.html        … 入口（級を選ぶトップページ）
5kyu/             … 5級教材（index.html + vocab.js）
4kyu/             … 4級教材（index.html + vocab.js）
.github/workflows … GitHub Pages 自動デプロイ
```

各級は素の HTML + CSS + JavaScript のみ（フレームワーク不要）。進捗はブラウザの localStorage に保存されます。
