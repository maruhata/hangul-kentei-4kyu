# ハングル能力検定 4級 合格教材 🇰🇷

ほぼ初心者から **ハングル能力検定試験 4級 合格** を目指す、ブラウザで動くインタラクティブ教材です。インストール不要、`index.html` を開くだけで使えます。

## 📖 使い方

- **PC**: `index.html` をダブルクリック
- **スマホ / どこでも**: GitHub Pages のURLを開く（下記）

進捗（覚えた単語・クイズ/模試の最高点・チェック）はブラウザに自動保存されます。

## ✨ 収録内容

| タブ | 内容 |
|---|---|
| 🏠 学習ガイド | 8ステップの学習計画＋進捗ダッシュボード |
| ㉠ 文字 | 母音・子音・濃音の反切表（タップで🔊音声＋読みの目安） |
| 🔊 発音 | 連音化・濃音化・鼻音化など、4級頻出の発音ルール |
| 📘 文法 | 26項目（5級復習＋4級新出）。例文に🔊読み上げ |
| 🗂 単語帳 | 約920語のフラッシュカード（分野別・音声つき） |
| ✏️ クイズ | 6分野の選択式クイズ（採点・自己ベスト保存） |
| 📝 模擬試験 | 本番形式：聞きとり40点＋筆記60点＝100点（60点で合格判定） |

## 🔊 音声について

ブラウザの音声読み上げ機能（Web Speech API, `ko-KR`）を使用しています。
韓国語の音声がお使いの端末に入っていない場合、再生されない・英語読みになることがあります。
Windows の場合は「設定 → 時刻と言語 → 言語 → 韓国語を追加（音声付き）」で改善します。

## 📁 ファイル構成

- `index.html` … 本体（文字・発音・文法・クイズ・模試）
- `vocab.js` … 語彙データ（約920語。追記で簡単に拡張できます）
